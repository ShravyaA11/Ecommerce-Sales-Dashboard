# Ecommerce-Sales-Dashboard
Developed an interactive E-Commerce Business Intelligence Dashboard in Power BI to analyze sales performance, customer behavior, product categories, profitability, and shipping operations. The dashboard provides actionable insights through KPI tracking, regional analysis, customer segmentation, and order performance monitoring to support data-driven business decisions.

# Project Objective
* Analyze overall sales and profit performance.
* Monitor customer purchasing behavior across regions.
* Identify top-performing product categories.
* Evaluate shipping efficiency and delivery performance.
* Support business decision-making through interactive visualizations.

# Key Features
* Interactive KPI Cards
* Sales & Profit Analysis
* Category-wise Performance Analysis
* Regional Sales Analysis
* Customer Segment Analysis
* Shipping & Delivery Tracking
* Dynamic Filtering using Slicers
* Drill-down Reporting
* Business Performance Dashboard

# Technologies Used
* Power BI
* Power Query
* DAX
* Data Modeling
* Data Cleaning & Transformation
* CSV Data Sources
* Business Intelligence Reporting

# Workflow
Data Collection → Data Cleaning → Data Transformation → Data Modeling → DAX Calculations → Dashboard Development → Business Analysis → Insight Generation → Interactive Reporting
